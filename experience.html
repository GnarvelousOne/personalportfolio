Professional Projects

  • COVID19 Tracker - This app uses Python and Beautiful Soup to scrape three pieces of data from worldometer.info: US COVID19 cases, deaths, and recoveries. It adds a timestamp to each one and writes each new entry to log.txt, deliminated with semi-colons. The app prints each statistic to the terminal as well as creates a tkinter GUI window displaying each statistic along with a 'Refresh' button to update the data. It also adds each new entry to a python list in order to plot graphs or other data science uses.

  • Billboard 100 Top #1 Hits of the 1990's - More practice web scraping using Beautiful Soup. Scrapes the Billboard 100's #1 hit songs from the 1990's in chronological order. First project utilizing jQuery. Final version will automatically scroll through each song without the need to click a button.

  • 2018 Colorado Go Code Challenge - A 72-hour competition where we formed teams, created a Minimum Viable Product that used public data to better inform business users, and gave a 10 minute presentation to a panel of judges. I formed a team that presented The Gnarvelous Growbrain, in concept mostly as the only code written at the time was the python tkinter water module. This competition was very beneficial as it was the brainstorming that led to the idea of networking growbrains, the connecting of farms to restaurants, and the tablet app for partnered restaurants to provide to dining room customers.

  • The Gnarvelous Growbrain - The original project which forced me to learn coding, this is an open-source app that uses the Raspberry Pi to transform the home gardener into a serious producer. Most functions are automated and monitored 24/7. Users can be linked together to create virtual co-op's. Restaurants can link to their local gardens for improved communication and provide a true "locavore" dining experience. Under Construction

  • Cheat Sheet - A simple Codecademy exercise in creating HTML tables, a place to put commonly used and important programming concepts relevant to my work and studies.

  • The Law of Thelema - My first project involving HTML and CSS. Thelema is an esoteric and somewhat spooky religion founded by Aleister Crowley in the early 1900s. Here I took a handful of their holy books and created an HTML study guide of sorts. This was the first project where I focused on creating a unified theme for a website and learned many of the basic HTML methods used. The style is meant to be semantic, old school HTML, reminiscent of early web design.

  • The New Testament - For the sacred sake of sanctimonious symmetry, I took the Thelema framework and used selected Christian New Testament works. A nice balance. Also a good exercise to learn how to make different websites based on the same framework. The amount of data entry is intense, so I might never actually finish it. Under Construction
